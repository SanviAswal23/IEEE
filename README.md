# 🎮 IEEE — The Game

A multi-level browser-based puzzle game built with React, where logic, code, and wit are your only weapons. Each level is a standalone challenge — solve them all if you dare.

---

## 🕹️ Levels

### 🕰️ Level 2 – The Clockwork Loop
[**Play →**](https://ieee-level2.vercel.app/)

An animated clock ticks in reverse, lighting up digits that players must decode in the right order. Crack the 6-digit code to unlock a coding challenge — but the loop resets if you're wrong.

Built with: React, CSS animations, custom digit input logic

---

### ♟️ Level 3 – Checkmate Files
[**Play →**](https://ieeel-level3.vercel.app/)

An 8×8 chessboard made entirely of `.txt` files hides a riddle. Only one file reveals the right path forward — wrong moves lead you deeper into confusion. ECHO, a poetic robot assistant, nudges you in the right direction without giving it all away.

Built with: React, dynamic grid rendering, conditional hint system

---

### 💣 Level 5 – Time Bomb
[**Play →**](https://ieee-level5.vercel.app/)

A 15-minute rapid-fire logic and coding quiz with no second chances. The clock is always running. One wrong answer and you feel it. Built with a full custom timer, question flow engine, and scoring system from scratch.

Built with: React, timer logic, stateful quiz flow

---

## 🤖 Meet ECHO

ECHO is the in-game robot assistant who appears across levels — offering cryptic hints, poetic nudges, and the occasional warning. She doesn't give answers. She makes you think harder.
